# JobFit ATS

An AI-powered ATS resume tailoring system.

## Features
- Auto-detects job programming intensity:
  - Red = Heavy programming (not suitable for fresher)
  - Yellow = Moderate (manageable with quick learning)
  - Green = Light (safe to apply)
- Auto-generates ATS-friendly resume (Template 1)
- Auto-creates practice projects for freshers
- Auto study-plan for moderate roles
- Initial + Post ATS score improvement
- Auto-extract company name
- Exports resume as: Prabhu_<Company>_resume.pdf
- 100% frontend (React) project
